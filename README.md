# XFeatTensorRT
A C++ TensorRT implementation of XFeat: Accelerated Features deep learning model for lightweight image matching by VeRLab: https://github.com/verlab/accelerated_features

## Requirements
1) OpenCV 4
2) CUDA 12.2
3) NVIDIA TensorRT 8.6 +
4) LibTorch
5) yaml-cpp
